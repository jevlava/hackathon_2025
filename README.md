## ğŸ’¡ The Solution Built
**FES (Flood Event Simulacrum)** is an intelligent route-planning system powered by machine learning.  
 
Our system:  
- Predicts potential **future flood events** in the Corpus Christi area.  
- Uses those predictions to **map out the safest and fastest travel routes**.  
- Alerts users when a **destination is not viable** due to high flood probability.  
 
This project bridges predictive modeling and real-world navigation, enabling safer and more informed travel decisions during flood events.
